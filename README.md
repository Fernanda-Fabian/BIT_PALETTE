# BIT_PALETTE
BitPalette es una aplicación móvil diseñada para revolucionar la gestión de colorimetría y el diseño visual. Está orientada a ayudar a diseñadores y desarrolladores a encontrar la armonía visual perfecta mediante la generación automática de paletas de colores con sus respectivos códigos hexadecimales.

## PROBLEMATICA  
Elegir los colores adecuados para un proyecto muchas veces es complicado y mas cuando te 
bloqueas buscando colores que combinen y esto hace perder mucho tiempo, mientras 
tratamos de buscar inspiración en la web. En el desarrollo de software la selección de una 
identidad visual siempre se dificulta para poder encontrar combinaciones de colores que 
respeten reglas de armonía visual durante el desarrollo de interfaces y esto dificulta el proceso 
de diseño y desarrollo de cualquier proyecto. 

## SOLUCIÓN 
Una app diseñada para generar paletas automáticas con sus códigos hexadecimales. La 
aplicación permite ver ejemplo de temas ya creados y tiene una sección de favoritos donde 
se guardan los colores que mas nos gustan. 
BitPalette es una solución para que cualquier usuario sin importar su profesión pueda 
encontrar y organizar su identidad visual de forma rápida. En lugar de elegir colores al azar, 
BitPalette ofrece un laboratorio que utiliza lógica de colorimetría para sugerir colores que 
garantizan la armonía visual.
